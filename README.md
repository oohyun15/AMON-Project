# AMON-Project

아주대학교 시리어스 게임제작팀 AMON의 전체 Unity 파일입니다.

아래 주의 사항을 읽은 후 작업해주시길 바랍니다.

## 주의 사항

0. **.gitignore 파일은 수정하지 않습니다.**

1. **자신이 작업한 파일만 Commit** 하도록 합니다. 절대로 프로젝트 전체 파일을 Commit하지 말아주세요.

2. 모든 작업은 **master**로 Commit하도록 합니다. 충돌 방지를 위해서 작업하실 때는 **새로운 임시 Scene을 만든 후** 작업하시길 권장합니다.

2. 버전 관리를 위해 Commit 시 Summary는 최대한 구체적으로 명시해주시길 바랍니다.

4. .모든 파일 수정 및 이동은 Unity 내에서 해 .meta 파일을 생성할 수 있도록 합니다.	불가피하게 그러지 못한 경우 반드시 Summary에 적고 이후 **해당 파일의 .meta 파일이 생성/Commit** 되어야 합니다.
