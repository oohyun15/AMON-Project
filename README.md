# AMON-Project

아주대학교 시리어스 게임제작팀 AMON의 전체 Unity 파일입니다.

아래 주의 사항을 읽어주시고 작업해주시길 바랍니다.

## 주의 사항

1. 자신이 작업한 파일만 Commit 하도록 합니다. **절대로 프로젝트 전체 파일**을 Commit하지 말아주세요.

2. 버전 관리를 위해 Commit 시 Summary는 최대한 구체적으로 명시해주시길 바랍니다.

3. 폴더 전체를 드래그해서 Commit 하지 않도록 합니다. 반드시 **파일 단위**로 옮겨주세요.

파일을 Commit할 때 해당 파일의 .meta 파일을 놓칠 수 있습니다.

4. .meta 파일 갱신 관련해 모든 파일 수정 및 이동은 Unity 내에서 할 수 있도록 합니다.

불가피하게 그러지 못한 경우 반드시 Summary에 적고 이후 **해당 파일의 .meta 파일이 생성/Commit** 되어야 합니다.
